# Prompthony

Convierte tus ideas en prompts perfectos para Veo, Kling, Seedance, Higgsfield, Midjourney y Nano Banana.

## Deploy

1. Sube este proyecto a GitHub
2. Importa el repo en Vercel
3. Agrega tu API key como variable de entorno: `ANTHROPIC_API_KEY`
4. Conecta tu dominio

Built with Next.js + Anthropic API.
